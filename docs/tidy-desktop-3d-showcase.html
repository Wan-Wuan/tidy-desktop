Organize / Launch / Search

一个面板，收住桌面的混乱。

这不是给桌面再套一层皮，而是缩短你每天重复的启动路径。分类、快捷键、图标缓存、文件发送和网页搜索都在一个节奏里。

分类像工作台一样清楚

创建分类与子分类，拖拽应用到合适位置。工具、资料、设计、游戏都能保留自己的入口。

搜索不用记完整名称

支持拼音、首字母、多关键词、路径和别名。输入 wx、vscode 或文件夹路径，都能快速抵达。

图标自己变整齐

自动提取 exe、lnk 和 Steam 游戏图标并缓存。刷新图标、清理失效项、导出诊断都在设置里。

文件发送更顺手

文档可复制到剪贴板，图片可拖拽或复制内容，适合微信、Word、QQ 等日常发送场景。

搜索引擎直接接入

输入前缀加空格即可切换 Bing、Google、GitHub、知乎、B 站等搜索入口。

维护能力留在手边

支持导入导出备份、扫描快捷方式、健康检查、打开数据目录和更新安装日志。

Daily Flow

整理一次，每天少找几分钟。

Tidy Desktop 的核心流程很短：先把入口收进来，再按你的工作习惯整理，最后交给快捷键和搜索。

> scan desktop shortcuts > group design tools > ctrl+k vscode > gh tidy desktop
01

扫描或拖入项目

从桌面、开始菜单或文件管理器导入应用、文件夹、图片、文档和 Steam 链接。

02

分类、排序、补图标

按工作流创建分类，拖拽排序。缺失图标会在后台补齐，保持首屏响应更轻。

03

用快捷键打开一切

主界面浏览分类，搜索框负责快速启动。Enter 打开，Ctrl + Enter 打开所在文件夹。
