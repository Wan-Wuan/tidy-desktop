核心功能

为每天都会打开的东西，做一个稳定入口

它不是简单把桌面图标搬到另一个地方，而是让应用、文件和游戏拥有可维护的分类、搜索和恢复能力。

分类整理

把应用、文件夹、图片和快捷方式按常用、工具、游戏、资料等分类管理，减少桌面堆叠。

快速搜索

通过名称、别名和关键词快速定位项目，支持常用优先与搜索框快捷动作。

图标修复

自动提取应用与快捷方式图标，刷新时保留可用缓存，避免图标刷新后变空。

文件与图片入口

不仅能管理应用，也能把常用文件夹、文档和图片加入启动列表，形成统一工作台。

自动更新

安装包可从 GitHub Release 获取，更新安装完成后会自动启动，减少手动操作。

备份与维护

提供备份、恢复、清理失效项、恢复隐藏项目等维护工具，让长期使用更放心。

使用方法

从安装到日常使用，只需要几个步骤

适合把桌面图标很多、常用应用很多、Steam 游戏很多的电脑整理成一个更清爽的入口。

下载安装

点击页面上的“下载最新版”，从 GitHub Release 获取 Windows 安装包。安装完成后打开 Tidy Desktop。

添加项目

把应用、文件夹、图片或快捷方式拖入应用，也可以通过添加入口选择本地路径。Steam 游戏会尽量读取对应信息。

整理分类

为项目设置分类、别名和常用状态。常用项目会在搜索和列表中更容易出现。

启动与定位

在搜索框输入名称、别名或关键词，回车打开项目；也可以通过分类标签浏览并启动。

刷新图标

当某些应用图标显示异常时，使用刷新图标功能。刷新过程会尽量复用缓存，并避免界面长时间卡住。

备份数据

在设置中导出备份，重装系统或迁移电脑时再导入。应用数据默认保存在当前 Windows 用户目录下。

快捷键与技巧

熟悉这些操作后，会更像一个真正的启动中心

快捷键以应用内实际设置为准；如果你修改过快捷键，请按自己的设置使用。

操作 用途
CtrlK 快速聚焦搜索框，直接输入应用或文件名称。
Enter 打开当前选中的搜索结果或列表项目。
Esc 关闭弹层、取消搜索或回到上一级操作。
Delete 对选中项目执行删除或移除操作，具体行为以界面提示为准。

搜索建议

给常用项目添加简短别名会明显提升搜索效率。比如给设计工具、浏览器、资料文件夹设置你最习惯输入的中文或拼音关键词。

  • 常用项目建议放在固定分类里。
  • 移动应用位置后，记得清理失效项或重新添加。
  • 图标异常时先刷新图标，再检查原始快捷方式是否有效。

常见问题

安装、更新和数据说明

这些说明适合在发布页面或项目首页里给新用户快速扫一遍。

页面为什么不显示版本号？

版本号容易在静态页面里过期，所以页面不写固定版本。下载按钮会优先读取 GitHub 最新 Release 的安装包地址，读取失败时仍会跳转到最新发布页。

更新后会自动打开吗？

安装更新完成后会自动启动应用。若系统拦截安装器或权限不足，请按 Windows 提示允许安装。

数据会上传到服务器吗？

项目列表、分类、图标缓存和设置主要保存在本机用户目录。你可以通过设置里的备份功能迁移数据。

为什么桌面快捷方式图标不同步？

Windows 有自己的图标缓存。安装或更新后如果桌面快捷方式仍显示旧图标，可以重新创建快捷方式，或等待系统刷新缓存。
