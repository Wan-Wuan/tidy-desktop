Chapter 1

不是换个图标架子，而是把启动流程变短

应用、文件、文件夹、游戏入口和搜索引擎都可以进入同一个工作流。需要的时候一按快捷键，不需要的时候它安静待在托盘。

多级分类管理

创建分类与子分类，拖拽调整顺序，把常用、工作、娱乐、文档分开放，桌面不再堆成一片。

快速搜索框

支持应用名、拼音、首字母、关键词和路径输入。独立搜索窗口更轻，适合随时唤起。

图标自动提取

从 exe、lnk、文件和 Steam 链接中提取图标并缓存，显示更稳定，刷新也更可控。

自动扫描导入

扫描桌面与开始菜单快捷方式，配合自动创建分类，初次整理不需要从零手动添加。

文件发送辅助

文档可复制到剪贴板，图片可拖拽或复制内容，适合发送到微信、Word、QQ 等场景。

更新与维护

支持检查更新、安装包下载、数据健康检查、诊断导出和数据目录打开。

Chapter 2

三步开始整理

它不强迫你改变习惯，而是把已有的桌面入口收拢起来。先导入，再分类，最后用快捷键打开。

1

导入桌面和开始菜单

使用扫描功能收集快捷方式，也可以手动添加应用、文件夹、文档和 Steam 游戏。

2

创建分类并拖拽整理

按用途拆分分类与子分类，拖拽应用到合适位置，让启动台像你的工作流一样排列。

3

用快捷键唤起和搜索

通过主窗口浏览分类，通过搜索窗口快速输入关键词。Enter 打开，Ctrl + Enter 打开所在文件夹。

FAQ

常见问题

它会替代 Windows 桌面吗？

不会。它更像一个整理过的启动台，帮助你减少桌面快捷方式和重复寻找应用的时间。

可以添加哪些类型？

可添加应用、快捷方式、文件夹、常见文档、压缩包、媒体、图片，以及 Steam 游戏链接。

图标不显示怎么办？

可以使用刷新图标和数据健康检查。图标会缓存到本地，避免每次打开都重新提取。

更新失败怎么办？

可进入 GitHub Releases 手动下载最新安装包。部分网络环境访问 GitHub 资源会不稳定，手动下载通常更可靠。
